# Notechat server

Server for notechat app made with Express and Socket.io

## Protocol

Socket.io

## Events

События, которые используются в протоколе


- [client] connect
- [server] session
- [server] users
- [server] past_history
- [server] user_connected
- [client / server] message
- [client] disconnect
- [server] user_disconnected
 

## Development

```
npm run dev
```

## Deployment

Deploy on heroku or where you want
